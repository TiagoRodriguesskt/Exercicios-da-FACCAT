# 🚀 Exercícios de Lógica de Programação - FACCAT

Este repositório contém minhas resoluções dos exercícios da apostila de lógica de programação da **FACCAT**, desenvolvidos em **Python**. 

O foco deste estudo é fortalecer os fundamentos de algoritmos, manipulação de variáveis e estruturas de controle, servindo como base para meus estudos em **Física Computacional**.

## 🛠️ Ferramentas Utilizadas
*   **VSCode**
*   **Linguagem:** Python 3.14.3.
*   

## 📂 Exercícios em Destaque
Aqui estão alguns dos problemas resolvidos:

*   **Cálculo Eleitoral:** Processamento de percentuais de votos válidos, brancos e nulos com validação de dados reais.
*   **Comissão de Vendas:** Cálculo de salário final com bônus por produtividade e porcentagem sobre vendas.
*   **Física (Em progresso):** Scripts para automação de cálculos e modelagem simples.

## 🚀 Como rodar os scripts
1. Certifique-se de ter o Python instalado.
2. Clone o repositório:
   ```bash
   https://github.com/TiagoRodriguesskt/Exercicios-da-FACCAT.git
   ```
3. Execute qualquer exercício:
   ```bash
   python nome_do_arquivo.py
   ```

---
💡 *Dica: Ao digitar valores de milhar nos scripts, não utilize pontos (ex: digite 1000 em vez de 1.000) para garantir a precisão dos cálculos decimais.*
